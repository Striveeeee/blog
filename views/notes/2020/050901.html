        webpack学习笔记-模块热替换

        Chan 2020-05-09 webpack

        模块热替换(hot module replacement 或 HMR)是 webpack 提供的最有用的功能之一。它允许在运行时更新所有类型的模块，而无需完全刷新。本页面重点介绍其实现，而 概念 页面提供了更多关于它的工作原理以及为什么它有用的细节。

        # 启用HMR

        由于我们已经在index.js中引用了show.js,因此删除show.js文件的入口起点,webpack.config.js:

        const path = require('path')
const HtmlWebpackPlugin = require('html-webpack-plugin')
const {
  CleanWebpackPlugin
} = require('clean-webpack-plugin')
const webpack = require('webpack')

module.exports = {
  mode: 'development',
  entry: {
    app: './src/index.js'
  },
  devtool: 'inline-source-map',
  devServer: {
    contentBase: './dist',
    hot: true
  },
  plugins: [
    ...
    new webpack.HotModuleReplacementPlugin()
  ],
  ...
}

        修改index.js:

        import Vue from 'vue'
import './style.css'
import dataXml from './data.xml'
import showMe from './show.js'

new Vue({
  el: '#app',
  data () {
    return {
      data: dataXml,
      showMessage: ''
    }
  },
  methods: {
    clickHandler () {
      this.showMessage = showMe()
    }
  },
  template: `
    <div class="hello">
      <h3>Hello Vue.js&webpack</h3>
      <p>{{ data.note.body[0] }}</p>
      <button @click='clickHandler'>show me</button>
      <p>{{ showMessage }}</p>
    </div>
  `
})

if (module.hot) {
  module.hot.accept('./show.js', function () {
    showMe()
  })
}

        # 通过 Node.js API

        使用Node.js API和利用express方法相似，在根目录新建dev-server.js:

        const webpackDevServer = require('webpack-dev-server');
const webpack = require('webpack');

const config = require('./webpack.config.js');
const options = {
  contentBase: './dist',
  hot: true,
  host: 'localhost'
};

webpackDevServer.addDevServerEntrypoints(config, options);
const compiler = webpack(config);
const server = new webpackDevServer(compiler, options);

server.listen(5000, 'localhost', () => {
  console.log('dev server listening on port 5000');
});

        # 使用vue-loader

        安装：

        npm install -D vue-loader vue-template-compiler
# or
yarn add vue-loader vue-template-compiler --dev

        修改webpack.config.js:

        const path = require('path')
const HtmlWebpackPlugin = require('html-webpack-plugin')
const {
  CleanWebpackPlugin
} = require('clean-webpack-plugin')
const webpack = require('webpack')
const VueLoaderPlugin = require('vue-loader/lib/plugin')

module.exports = {
  mode: 'development',
  entry: {
    app: './src/index.js'
  },
  devtool: 'inline-source-map',
  devServer: {
    contentBase: './dist',
    hot: true
  },
  plugins: [
    ...
    new VueLoaderPlugin()
  ],
 	...
  module: {
    rules: [
      ...
      {
        test: /\.vue$/,
        loader: 'vue-loader'
      }
    ]
  }
}
