          # 加载CSS

          引入css文件，需要安装style-loader和css-loader，并在module配置中添加这些loader：

          npm install --save-dev style-loader css-loader
# or
yarn add style-loader css-loader --dev
          1
          2
          3

          在webpack.config.js中添加：

          module: {
  rules: [
    {
      test: /\.css$/,
      use: [
        'style-loader',
        'css-loader'
      ]
    }
  ]
}

          # 加载 fonts 字体

          file-loader和url-loader可以接收并加载任何文件，然后将其输出到构建目录。因此我们也可以用它来处理字体文件。

          修改webpack.config.js:

          module: {
  rules: [
    ...,
    {
      test: /\.(woff|woff2|eot|ttf|otf)$/,
      use: [
      	'file-loader'
      ]
    }
  ]
}

          # 加载数据

          支持JSON是内置的，因此import Data from './data.json'是可以正常运行的。如果要使用CSV、TSV 和 XML格式的数据文件，可以使用csv.loader和xml-loader加载：

          npm install --save-dev csv-loader xml-loader
# or 
yarn add csv-loader xml-loader --dev
          1
          2
          3

          修改webpack.config.js:

          module: {
  rules: [
    ...,
    {
      test: /\.(csv|tsv)$/,
      use: [
      	'scv-loader'
      ]
    },
    {
      test: /\.xml$/,
      use: [
        'xml-loader'
      ]
    }
  ]
}
